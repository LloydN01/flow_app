## Installation

### Back-end Setup

1. **Navigate to the `backend` folder:**
    ```bash
    cd back-end
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

3. **Create a `.env` file:**
    ```env
    DB_USERNAME=your_db_name
    DB_PASSWORD=your_db_password
    ```

4. **Compile TypeScript:**
    ```bash
    npm run build
    ```
    or
    ```bash
    yarn build
    ```

### Front-end Setup

1. **Navigate to the `frontend` folder:**
    ```bash
    cd flow_app
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```
